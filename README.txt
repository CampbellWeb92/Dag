PAYPAL SETUP
Search for this placeholder in assets/script.js:
https://paypal.me/REPLACE/

Replace REPLACE with the breeder's PayPal.Me username.

IMPORTANT
A browser-only website cannot securely verify that a PayPal payment has completed. The included green "Demo: Confirm Successful Payment" button demonstrates the sold-status feature using localStorage. For a real marketplace, payment verification and permanent Sold status must be handled by a secure backend and PayPal webhooks/API.
